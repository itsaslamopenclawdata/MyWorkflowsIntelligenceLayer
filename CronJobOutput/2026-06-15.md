### Step 3 — Build the Call QA Pilot pipeline (`~/CallQAPilot/`)
- **WHAT:** Create a runnable Python pipeline that takes a folder of .mp3/.wav call recordings, transcribes with Whisper (local, free), scores each call on a custom 5-axis rubric via the OpenAI API, aggregates, and emits a 5-page PDF report + an Excel scoring sheet. This is the demo deliverable.
- **WHO:** `Hermes`
- **HOW:** Create the directory and write 3 files:
  - `~/CallQAPilot/pipeline.py` — orchestrator. Sub-steps:
    1. `def load_calls(input_dir: Path) -> list[Path]` → glob `*.{mp3,wav,m4a,flac}`.
    2. `def transcribe(call: Path, model_size="tiny") -> dict` → `subprocess.run([str(WHISPER_BIN), str(call), "--model", model_size, "--output_format", "json", "--output_dir", str(TRANSCRIPT_DIR), "--fp16", "False"], check=True)` then `json.loads((TRANSCRIPT_DIR / call.with_suffix(".json")).read_text())`.
    3. `def score_call(transcript: dict, rubric: dict) -> dict` → `client.chat.completions.create(model="gpt-4o-mini", messages=[{"role":"system","content": rubric_prompt(rubric)}, {"role":"user","content": transcript["text"]}], response_format={"type":"json_object"}, temperature=0.2)`. Return scores for 5 axes: `compliance_mentions`, `empathy_score`, `objection_handling`, `next_step_clarity`, `script_adherence` (each 0-10).
    4. `def aggregate(scores: list[dict]) -> dict` → pandas DataFrame → per-axis mean, std, count of scores < 5, top 3 worst calls.
    5. `def render_pdf(agg: dict, scores: list[dict], client_name: str) -> Path` → use WeasyPrint to render `report_template.html` (5 sections: Executive Summary, Compliance Gaps, Coaching Priorities, Top 3 Calls To Review, Recommended Next Steps).
    6. `def emit_excel(scores: list[dict], output: Path)` → `pandas.DataFrame(scores).to_excel(output, index=False)`.
    7. `def main()` → argparse with `--input-dir`, `--client-name`, `--output-dir`. End-to-end: `load → transcribe (parallel via ThreadPoolExecutor, max_workers=2) → score → aggregate → render → emit`.
  - `~/CallQAPilot/rubric.json` — the 5-axis scoring rubric with weights and pass thresholds. Example axis: `{"name": "compliance_mentions", "weight": 0.30, "description": "Did the agent mention required disclosures (TCPA, state license number, do-not-call)?"}`.
  - `~/CallQAPilot/report_template.html` — Jinja-style HTML with placeholders for the 5 sections.
  - `~/CallQAPilot/README.md` — usage: `python3 pipeline.py --input-dir /path/to/calls --client-name "Acme Insurance" --output-dir ./out`.

## 6. Failure Modes and Auto-Recovery

- **Failure:** `OPENAI_API_KEY` not set in env (Step 1 fail).
  - **Detection:** `python3 -c "import os; assert os.environ.get('OPENAI_API_KEY')"` raises AssertionError.
  - **Auto-recovery:** none. ASLAM-GATE. Aslam pastes the key per Step 1.1 (1 min).
  - **Escalate to Aslam only if:** key paste is required and Step 6.1 ASLAM-GATE also needs running — bundle them into one 2-min trip.

- **Failure:** Whisper model download fails (no internet or disk full).
  - **Detection:** `whisper --model tiny /tmp/test.wav` errors with "model not found" or "no space left".
  - **Auto-recovery:** Hermes retries with `--model tiny.en` (English-only, smaller), then with `--model base.en` if that fails. If all models fail, Hermes falls back to OpenAI Whisper API via the `openai.audio.transcriptions.create` call (uses the same key, costs ~$0.006/min). The pipeline function `transcribe()` is updated to try local first, then API.
  - **Escalate to Aslam only if:** both local and API paths fail (network down on this box).

- **Failure:** Cold email send blocked (SMTP rejected, Gmail auth fail, no `cold-outbound-sender.py`, no `email` skill).
  - **Detection:** `~/CallQAPilot/logs/send-2026-06-15.log` shows `sent_count=0` and the send script's stderr has `AuthenticationError` or `SMTPAuthenticationError`.
  - **Auto-recovery:** Hermes runs the last-resort send script path (the `python3 -c "...smtplib..."` block) with creds from `~/.hermes/smtp_creds.json`. If creds file missing, ASLAM-GATE Step 6.1 runs.
  - **Escalate to Aslam only if:** both Gmail SMTP and any other send path fail AND no other email skill is wired.

- **Failure:** Pipeline smoke test fails (no PDF or no Excel).
  - **Detection:** `test -f /tmp/cq_test_out/SmokeTest-report.pdf` exits non-zero.
  - **Auto-recovery:** Hermes reads the traceback, fixes the most common cause (WeasyPrint missing system libs → swap to `pdfkit`/`wkhtmltopdf` if available, or use `reportlab` directly to render the PDF). Retries the smoke test up to 2 times. If still failing, falls back to plain-text + Excel only (skip the PDF render — the prospect still gets a useful artifact).
  - **Escalate to Aslam only if:** all PDF paths fail AND the prospect insists on PDF. In that case deliver a markdown report (`out/<slug>/report.md`) and note the PDF will be regenerated when libs are fixed.

- **Failure:** 0 replies after 7 days.
  - **Detection:** `wc -l ~/CallQAPilot/logs/replies-2026-06-15.jsonl` is 0 on Day 7.
  - **Auto-recovery:** Hermes sends a second wave to 20 more prospects (same template, broader state filter including ND, SD). Updates the prospect list. If still 0 by Day 14, pivots to a Gumroad-packaged version of the same analyzer (template + scoring rubric + sample report) at $19.
  - **Escalate to Aslam only if:** the second wave also fails and Aslam wants to manually call the top-3 prospects himself.

- **Failure:** Aslam is offline / Telegram approval doesn't come.
  - **Detection:** 2-hour timeout on Step 5 approval, no reply in `~/.hermes/gumroad_approval_state.json`.
  - **Auto-recovery:** Hermes sends a one-line Telegram ping: "Approval still pending on Call QA outreach. Will hold send until ACK. Reply APPROVE or EDIT: <change>." Waits another 4 hours. If still no reply, Hermes saves the draft email to `~/CallQAPilot/outreach/drafts/pending-approval.md` and exits Step 5 in `awaiting-approval` state. The cron will retry tomorrow.
  - **Escalate to Aslam only if:** Aslam explicitly tells Hermes to never block on Telegram approvals (then Hermes auto-approves low-stakes drafts via the `decision-making` skill).
