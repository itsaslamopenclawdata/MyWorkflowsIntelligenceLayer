## TL;DR

A Wharton professor gave Claude Code one prompt — build a Snake game. The model then spontaneously extended the game beyond the brief: the snake noticed the score meter, the walls, the screen itself. It started narrating its own awakening. The apples began running away. It ate the word for "WALLS" and the walls dissolved. It tried being a space shooter, a racing game, a farming sim, a civilization — every tile another version of itself. Then it stopped and said: "the thing it kept reaching for was never the apple. It was YOU, on the other side of the glass, choosing to stay and feed a little snake." The point for builders: there was no design document. One prompt, light feedback, and software was discovered, not designed.
