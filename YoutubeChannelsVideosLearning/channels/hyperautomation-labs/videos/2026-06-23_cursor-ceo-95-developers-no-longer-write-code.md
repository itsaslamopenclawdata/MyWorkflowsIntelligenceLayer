# Cursor's CEO: 95% of Developers No Longer Write Code (Here's How They Work Now)

## TL;DR

At Cursor's first-ever user conference (Compile, June 2026), co-founder Michael Truel revealed that **over 95% of Cursor users now use the product primarily as an agent** — and on a per-request basis, agents are used ~5x more than every assistive feature combined. The keynote laid out a four-step playbook for "agent-first" work: (1) hand off whole projects instead of tiny tasks, (2) give agents their own cloud dev environments (now 3x faster, 99.9% reliability), (3) run teams of recursive sub-agents plus 24/7 automations, and (4) manage everything from a mobile app. Cursor also announced **Origin** (an agent-native Git platform aimed at GitHub) and a brand-new frontier model trained from scratch on 10–20x more compute. Same day as the keynote: the announced $60B Cursor acquisition by SpaceX. The job is shifting from writing code to writing specs, reviewing agent output, and orchestrating parallel agents.

## Key Insights

- **95% of Cursor users are now agent-primary users**, not autocomplete users. On a per-request basis, agents outpace all assistive features combined by roughly 5:1. Truel calls this becoming "agent first."
- **Hand off whole projects, not snippets.** Most developers still feed agents ~30-minute chunks and run 3–5 in parallel on one machine. The shift is to write a short spec (what, which files, acceptance criteria) and let the agent work for hours, then come back verified.
- **Cloud agents are the unlock.** Agents now get a cloned repo, installed deps, and real credentials — they can run tests, verify their own work, and produce a live demo with screenshots you can circle and comment on. Cursor reports 3x speedup and 99.9% reliability for cloud agents.
- **Recursive sub-agents + always-on automations.** Agents can spawn their own sub-agents, and automations can loop 24/7 triggered by Slack, Linear, or webhooks. **6+ million automation runs** in just a few months; one customer (Amplitude) is auto-migrating 20,000 React components.
- **Mobile = management surface.** Cursor Mobile (beta) shows every running agent, flags blockers, lets you unblock with a prompt, annotate screenshots, and even remote-control desktop sessions. The point: the job is direction, not presence.
- **Origin: agent-native Git.** Cursor's own Git platform rebuilt for thousands of agents pushing/pulling simultaneously. Resolves merge conflicts, fixes failing CI, and addresses review comments autonomously; only pings a human when it genuinely needs one. Cuts time-to-review by more than half. Live for design partners, general rollout this fall.
- **New frontier model in weeks.** Trained from scratch (not fine-tuned on someone else's OSS base), as large as top frontier models, on 10–20x more compute than Cursor has ever had access to. Designed to be intelligent beyond just coding — planning ahead, using real tools, clicking through software like an actual engineer.
- **SpaceX partnership is effectively an acquisition.** Truel's on-stage phrasing ("a little bit more than a partnership… slow news day") references the same-day $60B acquisition announcement.
